

Ada Library for OS Dependent Tasks



    This library contains some packages that are meant to easy the develop-
ment of cross-platform softwares in Ada.
    It also contains some usefull libraries for string manipulation and a
a package for handling environment variables while the GNAT GCC doesn't
come with Ada.Environment_Variables package in Gentoo Linux.

   It's released by the GNAT Modified GPL License (see LICENSE.txt and GPL.txt).

   To build simply run ./build.sh. If you have >=gnat-gcc-4 it should compile everything and put the objects inside lib and the executables inside bin
